### Markdown

マークダウン形式は、あなたが入力した文字をそのままスタイリングしてくれる、簡単で使いやすい構文です。
```markdown
Syntax highlighted code block

# Header 1
## Header 2
### Header 3

- Bulleted
- List

1. Numbered
2. List

**Bold** and _Italic_ and `Code` text

[Link](url) and ![Image](src)
```
↑コードを強調するための構文らしい。
